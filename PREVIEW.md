# מצב תצוגה מקדימה (ללא מסד נתונים)

כדי לראות את כל הפלואו של ה-CRM **בלי Supabase ובלי התחברות**:

1. צור קובץ `.env.local` בשורש הפרויקט עם:
   ```
   NEXT_PUBLIC_PREVIEW_MODE=true
   ```

2. הרץ:
   ```bash
   npm run dev
   ```

3. פתח בדפדפן — תועבר אוטומטית לרשימת התיקים. כל הנתונים הם להמחשה (mock).

**מה עובד במצב תצוגה מקדימה:**
- כניסה ישירה ל־/cases (בלי מסך התחברות)
- תפריט מלא: תיקים, סגירה, אישורים, תוספות, התראות
- צפייה בתיקים, פרטי תיק, צ'קליסט, סגירה, אישורי CEO, תוספות, התראות
- כפתורים ופעולות לא שומרים למסד — רק מציגים את הממשק

**כדי לחזור למצב רגיל (עם Supabase):**  
הסר את `NEXT_PUBLIC_PREVIEW_MODE` מ־`.env.local` או הגדר ל־`false`, והגדר `NEXT_PUBLIC_SUPABASE_URL` ו־`NEXT_PUBLIC_SUPABASE_ANON_KEY`.
